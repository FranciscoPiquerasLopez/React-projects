# GALERÍA DINÁMICA

## Descripción:
- Búsqueda de imágenes usando una API + uso de debounce para evitar por cada tecla
pulsada búsquedas innecesarias.

## Instalación:
Sigue estos pasos:
1. **pnpm install**
2. **pnpm run dev**

## Características:
- Ha sido realizado con React y JavaScript + SWC.
